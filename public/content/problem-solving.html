Bài này học gì? Học cách phân tích bài toán: hiểu đề → chia nhỏ → thử ví dụ → pseudo-code → code.

1. Đọc đề như checklist

  • Input là gì?
  • Output là gì?
  • Ràng buộc/điều kiện đặc biệt?
  • Tự tạo 1–2 ví dụ để kiểm tra hiểu đúng

2. Chia nhỏ bài toán

Chia bài toán thành các bước nhỏ, mỗi bước kiểm chứng được.

Bước 1: ...
Bước 2: ...
Bước 3: ...

3. Thử tay bằng ví dụ

Chạy thử bằng tay với input nhỏ để biết output đúng phải là gì.

4. Pseudo-code trước

đặt count = 0
duyệt từng phần tử x
  nếu x chẵn thì count++
in count

Ôn lại nhanh

  • Hiểu đề trước khi code
  • Chia nhỏ + ví dụ
  • Pseudo-code để tránh “nhảy cóc”
