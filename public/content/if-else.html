🎯 Bài này học gì? Học cách bảo máy tính: "Nếu điều kiện đúng thì làm A, nếu sai thì làm B." Giống như khi mẹ nói: "Nếu trời mưa thì mang áo mưa, không thì mang nón."

1️⃣ Hiểu "đúng" và "sai"

Máy tính chỉ hiểu hai thứ: đúng (✅) và sai (❌).

Ví dụ

  • "3 lớn hơn 2 không?" → ✅ Đúng
  • "5 bằng 10 không?" → ❌ Sai
  • "Bạn có ít nhất 1 mạng không?" → ✅ Đúng hoặc ❌ Sai

Khi ta viết if-else, ta đang nói với máy: "Hãy xem điều kiện này đúng hay sai. Đúng thì làm việc này, sai thì làm việc kia."

2️⃣ Hình dung: con đường rẽ đôi

Hãy tưởng tượng bạn đứng trước hai con đường:

🔀 If-else = đứng ở ngã ba, kiểm tra bảng chỉ dẫn (điều kiện). Đúng thì đi đường A, sai thì đi đường B.

3️⃣ Viết ra sao?

Trong code, ta viết gần giống tiếng Anh:

if (điều kiện đúng?) {
  làm việc A;
} else {
  làm việc B;
}

Nhớ 💡 Nhớ nhé: if = "nếu", else = "còn không thì". Máy sẽ chỉ chạy một trong hai nhánh thôi (A hoặc B), không chạy cả hai.

4️⃣ Ví dụ dễ: Điểm đạt hay chưa?

Quy định: điểm từ 5 trở lên là đạt, dưới 5 là chưa đạt.

Ví dụ

Bước 1: Có biến điểm, ví dụ diem = 7.

Bước 2: Hỏi: "diem có lớn hơn hoặc bằng 5 không?"

Bước 3: Nếu có → in "Đạt" 🎉. Nếu không → in "Chưa đạt" 📝.

if (diem >= 5) {
  in("Đạt");
} else {
  in("Chưa đạt");
}

Thử trong đầu: nếu diem = 3 thì máy in gì? → In "Chưa đạt", vì 3 < 5 (điều kiện sai).

5️⃣ Nhiều nhánh: else if

Đôi khi không chỉ có hai lựa chọn mà có ba, bốn:

  • Điểm từ 8 trở lên → "Giỏi" 🌟
  • Từ 5 đến dưới 8 → "Khá" 👍
  • Dưới 5 → "Cần cố gắng" 💪

Ta dùng else if (còn không thì nếu...):

if (diem >= 8) {
  in("Giỏi");
} else if (diem >= 5) {
  in("Khá");
} else {
  in("Cần cố gắng");
}

Tip 🧩 Máy đọc từ trên xuống. Gặp điều kiện đúng đầu tiên thì làm xong rồi bỏ qua các dòng còn lại.

6️⃣ Trong game dùng if-else thế nào?

Ví dụ:

  • Nếu chạm quái → trừ 1 mạng, không thì không trừ.
  • Nếu số mạng = 0 → hiện màn hình "Game Over", không thì tiếp tục chơi.
  • Nếu nhân vật đứng trên đất → mới cho nhảy, không thì không nhảy (tránh nhảy lia lịa trên không).

🎮 Tóm lại: if-else giúp game "biết" phản ứng khác nhau tùy tình huống (đúng/sai).

📌 Ôn lại nhanh

  • If-else = "Nếu đúng thì làm A, sai thì làm B."
  • Máy chỉ chạy một trong hai nhánh.
  • Dùng else if khi có nhiều hơn hai lựa chọn.
  • Trong game: dùng để kiểm tra điều kiện (chạm quái, hết mạng, đứng đất...) rồi làm hành động tương ứng.
